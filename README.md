## 🚀 Overview

The **AI Code Debugger** (Nexus Engine) is a premium code analysis platform that leverages Groq's ultra-low-latency inference to instantly identify bugs, optimize performance, and output fully refactored, production-ready frontend code. 

Engineered with a strict focus on web technologies, this platform features a split-screen IDE layout inspired by top-tier professional software, ensuring high-accuracy diagnostics without the noise of unrelated programming languages.

## ✨ Key Features

*   **Side-by-Side IDE Workspace:** A professional, split-screen desktop layout (Code Input on the left, AI Output on the right) that maximizes screen real estate and completely eliminates "deep scrolling" issues.
*   **Intelligent Refactoring:** Instantly analyzes HTML, CSS, and JS to detect syntax errors and logic flaws, providing clean, optimized code blocks.
*   **Guaranteed Full-Code Generation:** The AI engine is strictly instructed to output 100% complete, ready-to-use code. It never truncates output or uses frustrating placeholders like `// rest of the code`.
*   **Strict Web-Only Scope:** An intelligent filtering system automatically rejects non-web languages (Python, C++, Java, etc.), keeping the engine laser-focused on frontend development.
*   **Premium Corporate UI:** Features a high-end, minimalist deep-slate aesthetic crafted using strictly grayscale and blue-slate tones.
*   **Frictionless UX:** Includes smooth animations, dynamic markdown rendering, and one-click interactive copy buttons for all AI-generated code.

## 🛠️ Technology Stack

*   **Frontend Core:** HTML5, CSS3, Vanilla JavaScript (ES6+)
*   **Framework:** Bootstrap 5 (Responsive Grid, UI Components, Icons)
*   **Markdown Parsing:** Marked.js
*   **AI Integration:** Groq API (`llama-3.3-70b-versatile`)

## ⚙️ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/MASTER870-CMD/AI-CODE-debugger.git](https://github.com/MASTER870-CMD/AI-CODE-debugger.git)
    cd AI-CODE-debugger
    ```

2.  **Configure your API Key:**
    Open the `index.html` file in your preferred code editor. Locate the JavaScript configuration section near the bottom of the file and insert your Groq API key:
    ```javascript
    const GROQ_API_KEY = "gsk_your_api_key_here"; 
    ```

3.  **Launch the Application:**
    Since this is a client-side application, no build steps are required. Simply open `index.html` in any modern web browser or serve it via a local development server (like VS Code Live Server).
